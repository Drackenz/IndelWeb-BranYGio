## ¿Qué diferencia hay entre display: grid y display: flex?

Grid sirve para organizar los elementos en filas y columnas, por lo que es ideal para crear la estructura principal de una página. Flexbox organiza los elementos en una sola dirección, ya sea horizontal o vertical, por lo que es muy útil para menús, botones o grupos de elementos.

## ¿Cuándo usarías cada uno?

Usaría Grid para distribuir el contenido principal de una página y Flexbox para acomodar elementos como el menú de navegación, tarjetas o formularios.
