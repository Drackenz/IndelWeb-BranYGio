## ¿Qué es una API REST?

Una API REST es un servicio que permite que diferentes aplicaciones intercambien información por medio de Internet usando peticiones HTTP.

## ¿Qué significa el código 200?

Significa que la solicitud fue exitosa y los datos fueron obtenidos correctamente.

## ¿Qué significa el código 404?

Significa que el recurso solicitado no fue encontrado, por ejemplo un Pokémon que no existe.

## ¿Qué significa el código 500?

Significa que ocurrió un error interno en el servidor y este no pudo procesar la solicitud.
