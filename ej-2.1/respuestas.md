## ¿Por qué usamos htmlspecialchars()?

Se utiliza para convertir caracteres especiales en texto seguro. De esta forma se evita que alguien pueda insertar código HTML o JavaScript malicioso dentro del formulario.

## ¿Qué ataque previene?

Principalmente ayuda a prevenir ataques de Cross Site Scripting (XSS), donde un usuario intenta ejecutar código malicioso dentro de una página web.
