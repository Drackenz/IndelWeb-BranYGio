## ¿Qué diferencia hay entre <article> y <section>?

La etiqueta `<section>` se utiliza para dividir una página en diferentes secciones relacionadas con un tema. En cambio, `<article>` se usa para contenido independiente que puede entenderse por sí solo, como una noticia, un proyecto o una publicación.

## ¿Cuándo usarías cada una?

Usaría `<section>` para organizar las partes de una página, por ejemplo educación, habilidades o contacto. Usaría `<article>` cuando el contenido pueda existir de forma independiente, como un proyecto, un blog o una noticia.
