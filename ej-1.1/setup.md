## ¿Qué hace el atajo ! + Tab?

El atajo `! + Tab` utiliza Emmet para generar automáticamente la estructura básica de un documento HTML5. Esto permite ahorrar tiempo y empezar a trabajar sin escribir todo el código desde cero.

## ¿Por qué Live Server recarga automáticamente?

Live Server detecta cuando se guardan cambios en los archivos del proyecto y actualiza automáticamente la página en el navegador. Así se pueden ver los cambios de inmediato sin tener que recargar manualmente.
